# Images Folder

This folder contains all images for the TLS Commercial Parts website.

## Required Images:

### 1. Hero Section
- **hero-bg.jpg** (1920x1080px or larger)
  - Professional image showing motor parts or Mercedes-Benz vehicles
  - Should be high quality and well-lit
  - Consider images of: workshop, car parts display, or Mercedes-Benz in service

### 2. Logo (Optional)
- **logo.png** (300x100px recommended)
  - Company logo with transparent background
  - Can be used to replace the text logo in navigation

### 3. Products Section (Coming Soon)
- Product images for motor parts
- Mercedes-Benz specific parts images

### 4. About Us Section (Coming Soon)
- Team photos
- Workshop/facility images

## Image Sources:
- Use high-quality stock photos from:
  - Unsplash.com (free)
  - Pexels.com (free)
  - Your own professional photos

## Image Optimization Tips:
- Compress images before uploading
- Use WebP format for better performance
- Maximum file size: 500KB per image
- Use descriptive filenames (e.g., mercedes-engine-parts.jpg)

## Current Status:
- The hero section uses a gradient background as fallback
- Add hero-bg.jpg to display a custom background image
